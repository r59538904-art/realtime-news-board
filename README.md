# 世界ニュース速報 — リアルタイム経済ニュースボード

Bloomberg・NHK・東洋経済オンライン・Nikkei Asia・ITmediaなど、複数の公開RSSフィードを集約して自動更新するニュースボードです。サーバー不要の単一HTMLファイルで、ブラウザで開くだけで動作します。

## 主な機能

- Bloomberg(市況・経済・テック)、NHK、東洋経済オンライン、Nikkei Asia、ITmedia、日経クロステック、CNET Japan、MarketWatch など約20の公開RSSフィードを集約
- 1分ごとに自動更新(過去2日以内の記事のみ表示)
- 英語記事はMyMemory / Google翻訳で自動的に日本語見出しへ変換(原文も併記)
- 株価・AI・企業関連キーワードによるトピック絞り込み(ON/OFF切替可)
- 見出し・要約の頻出語による簡易センチメント表示(▲/▼)、NEWバッジ
- ソース別フィルター・キーワード検索
- TradingViewティッカーで為替・株価指数もあわせて表示

## 使い方

`index.html` をブラウザで直接開くか、GitHub Pages で公開したページにアクセスするだけです。インストールやビルドは不要です。
